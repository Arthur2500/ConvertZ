# ConvertZ
A video conversion webapp using ffmpeg, 100% written by ChatGPT (https://chatgpt.com/share/1b825f58-7a4d-487e-a9e7-c53e56390b01)

## How to run:
```docker run -d -p 5000:5000 ghcr.io/YOUR_GITHUB_USERNAME/convertz:latest```
